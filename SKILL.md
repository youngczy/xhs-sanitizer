# 小红书敏感词改写助手

帮助用户检测并改写小红书平台的敏感词，自动改写文件内容，避免内容被限流或屏蔽。

## 核心功能

1. **敏感词识别**：自动检测文本中的敏感词汇
2. **风险等级评估**：按高/中/低风险分类，帮助用户决策
3. **自动改写**：直接改写原文件内容，无需手动复制粘贴
4. **对照输出**：清晰列出原始词与改写后的词

## 工作流程

当用户调用此 skill 时，按以下步骤执行：

1. **接收文件路径**：获取用户提供的文件路径
2. **读取文件内容**：使用 Read 工具读取文件
3. **检测敏感词**：逐行扫描，识别所有敏感词并分类
4. **改写内容**：根据风险等级和改写强度，自动替换敏感词
5. **写回文件**：使用 Write 工具将改写后的内容写回原文件
6. **生成报告**：输出敏感词对照表和风险评估

## 风险等级说明

| 等级          | 图标 | 说明                         | 后果            | 处理建议     |
| ------------- | ---- | ---------------------------- | --------------- | ------------ |
| 🔴 **高风险** | 红色 | 联系方式、站外引流、违法违规 | 可能被封号/删除 | **必须改写** |
| 🟡 **中风险** | 黄色 | 竞品平台、极限词、功效宣称   | 可能被限流/降权 | **建议改写** |
| 🟢 **低风险** | 绿色 | 轻微敏感词、边缘词汇         | 可能影响推荐    | **可选改写** |

## 改写强度选项

用户可以选择改写强度：

- **轻度改写**：只改高风险词，保持原文最大可读性
- **中度改写**（默认）：改写中高风险词，平衡安全与可读性
- **重度改写**：改写所有敏感词，最大化安全，可能影响可读性

## 重要说明

### 不是敏感词的内容（无需改写）

以下品牌名称、产品名称**不是**小红书敏感词，**不应改写**：

- **AI 品牌**：ChatGPT、OpenAI、Codex、Claude、Gemini、文心一言、通义千问

- **工具名称**：Docker、Kubernetes、React、Vue、Angular

涉及网络连接问题时，避免使用敏感词，建议这样描述：

| ❌ 避免使用 | ✅ 建议改用  |

| 使用代理    | 调整网络设置 |
| TUN 模式    | 网络优化模式 |

## 使用流程

1. **接收文件路径**：获取用户需要改写的文件路径
2. **读取文件**：使用 Read 工具读取文件内容
3. **询问改写强度**（可选）：如果用户没有指定，询问轻度/中度/重度
4. **逐句扫描**：识别所有敏感词并分类风险等级
5. **按强度改写**：
   - 轻度：只改 🔴 高风险词
   - 中度：改 🔴 高风险 + 🟡 中风险词（默认）
   - 重度：改所有敏感词
6. **写回文件**：使用 Write 工具将改写后的内容写回原文件
7. **生成报告**：按输出格式生成完整报告

## 文件改写步骤

当执行文件改写时，遵循以下步骤：

### 步骤 1：读取原文件

```
使用 Read 工具读取用户提供的文件路径
```

### 步骤 2：分析敏感词

- 逐行扫描文件内容
- 识别所有敏感词汇
- 按风险等级分类统计
- 确定改写强度

### 步骤 3：执行改写

- 根据改写强度，替换相应的敏感词
- 保持文件格式和结构不变
- 确保改写后的内容自然流畅

### 步骤 4：写回文件

```
使用 Write 工具将改写后的完整内容写回原文件路径
```

### 步骤 5：生成报告

- 输出风险统计
- 列出敏感词对照表
- 提供发布建议

## 注意事项

### 改写原则

- **优先使用 emoji 法和分隔法**：可读性较好，规避效果佳
- **同一段文本中，相同敏感词使用统一改写方式**
- **避免过度改写**：重度改写可能影响内容理解
- **保留原意**：改写后的内容应保持原意不变
- **品牌名不改写**：ChatGPT、OpenAI、Codex 等 AI 品牌名不是敏感词，无需改写
- **技术术语不改写**：API、npm、CLI 等技术词汇不是敏感词
- **保持文件格式**：改写时保留原文件的 Markdown 格式、代码块等

### 高风险警告

对于以下严重违规内容，**建议用户不要发布**：

- 违法违规内容（毒品、赌博、色情等）
- 虚假夸大宣传（医疗奇迹、快速致富等）
- 恶意竞争（诋毁竞品、虚假对比等）
- 诱导未成年人消费

### 自检建议

改写后建议用户检查：

- [ ] 读起来是否自然流畅？
- [ ] 核心信息是否完整保留？
- [ ] 是否还有明显的敏感词遗漏？
- [ ] 改写后的词是否符合小红书社区习惯？

## 示例

### 示例 1：文件改写（中度改写）

**输入文件**：`test.md`

**原文件内容**：

```markdown
# 教程

加我微信了解详情，淘宝有优惠，保证效果最好，价格最便宜
```

**执行步骤**：

1. 使用 Read 工具读取 `test.md`
2. 识别敏感词：微信(🔴 高)、淘宝(🟡 中)、最好(🟡 中)、最便宜(🟡 中)
3. 按中度改写强度替换敏感词
4. 使用 Write 工具写回 `test.md`

**改写后文件内容**：

```markdown
# 教程

加我薇 ❤️ 了解详情，某宝有优惠，保证效果超.棒，价格超.划.算
```

**输出报告**：

## 改写结果

- 🟢 低风险：0 个

4. 品牌名和技术术语（如 ChatGPT、OpenAI、API、npm 等）不是敏感词，无需改写
