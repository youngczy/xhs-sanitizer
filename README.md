# xhs-sanitizer

小红书敏感词检测与自动改写工具 - 让你的内容安全过审，避免限流和屏蔽。

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Claude Code](https://img.shields.io/badge/Claude%20Code-Skill-green.svg)](https://claude.ai)

## 功能介绍

xhs-sanitizer 是一个专门用于小红书平台的敏感词检测与改写工具。它可以自动识别文案中的敏感词汇，并根据风险等级提供相应的改写建议，帮助你的内容顺利通过平台审核。

### 核心功能

- **敏感词识别**：自动检测 13 大类敏感词汇
- **风险等级评估**：按 🔴 高/🟡 中/🟢 低风险分类
- **自动改写**：直接改写原文件内容，无需手动复制粘贴
- **改写强度选择**：轻度/中度/重度三档可选
- **详细报告**：输出敏感词对照表和风险评估

## 安装方法

### 前置要求

- [Claude Code](https://claude.ai/code) 已安装
- Claude Code 版本 >= 0.40.0

### 安装步骤

1. 克隆或下载本仓库到本地：

```bash
git clone https://github.com/yourusername/xhs-sanitizer.git
```

2. 将 skill 复制到 Claude Code 的 skills 目录：

```bash
# macOS/Linux
cp -r xhs-sanitizer ~/.claude/skills/

# Windows
xcopy /E /I xhs-sanitizer %USERPROFILE%\.claude\skills\
```

3. 重启 Claude Code，skill 将自动加载

## 使用方法

### 基本使用

在 Claude Code 中输入以下命令：

```
/xhs-sanitizer /path/to/your/file.md
```

### 工作流程

1. **提供文件路径**：告诉 Claude 你要检测的文件路径
2. **选择改写强度**：
   - **轻度改写**：只改高风险词，保持原文最大可读性
   - **中度改写**（默认）：改写中高风险词，平衡安全与可读性
   - **重度改写**：改写所有敏感词，最大化安全
3. **自动改写**：Claude 会自动读取、改写并保存文件
4. **查看报告**：获取详细的敏感词对照表和风险评估

### 示例

**输入文件** (`example.md`):
```markdown
# 美妆推荐

大家好！今天推荐一款超好用的产品。
加我微信咨询：wx123456
淘宝店铺最便宜，效果最好！
能让你快速美白，一个月见效。
```

**执行命令**:
```
/xhs-sanitizer /path/to/example.md
```

**输出结果**:
```markdown
## 改写结果

### 风险统计

- 🟡 中风险：4个（最好、美白、见效）
- **总体风险等级**：高
- **建议改写强度**：中度
```

## 改写策略

工具提供多种改写策略，确保内容既能通过审核，又保持可读性：

| 策略 | 说明 | 示例 |
|------|------|------|
| **emoji法** | 用 emoji 替代或配合 | 微信 → 薇❤️ |
| **模糊法** | 用"某"字替代 | 淘宝 → 某宝 |
| **分隔法** | 用符号分隔字符 | 微信 → 微.信 |
| **替换法** | 用近义词替代 | 最好 → 超棒 |
| **拼音法** | 使用拼音或首字母 | 支付宝 → Z.F.B |
| **代称法** | 使用行业代称 | 医生 → 👨‍⚕️ |

## 注意事项

### 改写原则

- **优先使用 emoji 法和分隔法**：可读性较好，规避效果佳
- **同一段文本中，相同敏感词使用统一改写方式**
- **避免过度改写**：重度改写可能影响内容理解
- **保留原意**：改写后的内容应保持原意不变

### 不是敏感词的内容

以下品牌名称、产品名称**不是**小红书敏感词，**不应改写**：

- **AI 品牌**：ChatGPT、OpenAI、Codex、Claude、Gemini
- **技术术语**：API、Node.js、npm、CLI、IDE、VS Code、GitHub
- **编程语言**：Python、JavaScript、Java、C++、Go、Rust
- **工具名称**：Docker、Kubernetes、React、Vue、Angular

### 高风险警告

对于以下严重违规内容，**建议不要发布**：
- 违法违规内容（毒品、赌博、色情等）
- 虚假夸大宣传（医疗奇迹、快速致富等）
- 恶意竞争（诋毁竞品、虚假对比等）
- 诱导未成年人消费
